Personal
Gestión del equipo humano del taller. Cubre desde la configuración de accesos hasta el seguimiento del desempeño.

- Perfil de empleados y técnicos — Ficha de cada integrante del equipo con datos personales, rol, sucursal asignada y fecha de ingreso.
  
- Roles y permisos por sucursal — Control de qué puede ver y hacer cada usuario dentro del sistema, diferenciado por sede si corresponde.
  
- Asignación de OTs por técnico — Distribución de órdenes de trabajo entre los técnicos disponibles. Vista de carga por persona para balancear el trabajo.
  
- Control de asistencia — Registro de entrada y salida de cada empleado. Base para el cálculo de horas trabajadas.
  
- Liquidación de sueldos básica — Cálculo de haberes en base a horas trabajadas, categoría y adicionales.
  No reemplaza un sistema de RRHH completo pero cubre las necesidades básicas de un taller.
- Productividad y carga horaria — Métricas por técnico: OTs completadas, tiempo promedio de reparación, ingresos generados.
  Útil para bonificaciones o evaluación de desempeño.
